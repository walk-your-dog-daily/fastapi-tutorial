# FastAPI Starter Template

This is a simple starter project for building APIs with FastAPI

## Tech Stack

- **Backend**: FastAPI
- **Server**: Uvicorn (ASGI server)
- **Environment**: Python 3.7+
- **Optional**: SQLModel / SQLAlchemy, PostgreSQL, Docker
